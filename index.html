03. DealChecker: AI 계약서 공정성 분석 플랫폼
+
2025 대전광역시 교육발전특구 고교–대학 연계 사업

1. 프로젝트 개요

현대 사회에서 계약서는 주거, 고용, 금융 등 일상 전반에 깊이 관여하고 있지만, 전문적인 법률 지식이 없는 일반 사용자가 계약서의 불공정 조항이나 잠재적 위험 요소를 스스로 식별하기는 어렵습니다. 특히 청년층의 경우 정보 접근성과 법률 이해도의 한계로 인해 불리한 계약을 체결하는 사례가 반복되고 있습니다.

본 프로젝트 DealChecker는 이러한 문제를 해결하기 위해, AI 기반 자연어 처리 기술을 활용하여 계약서를 자동 분석하고 불공정 조항을 시각적으로 제공하는 웹 플랫폼을 개발하는 것을 목표로 진행되었습니다.

나아가 단순 분석에 그치지 않고, 개인 계약 이력을 체계적으로 관리할 수 있는 기능을 포함함으로써 실질적인 법률 지원 도구로 확장 가능성을 제시하였습니다.

2. 개발 동기 및 목적

2.1 개발 동기

  • 중앙화된 법률 정보 구조로 인해 일반 사용자는 계약의 위험성을 스스로 판단하기 어려움
  • 불공정 조항은 존재하더라도, 이를 인지하지 못한 채 계약이 체결되는 구조적 문제
  • 특히 청년층을 중심으로 법률 서비스 접근성의 격차가 사회 문제로 대두

2.2 개발 목적

  • AI 기반 계약서 분석 기능을 통해 법률 지식이 부족한 사용자도 계약 내용을 이해할 수 있도록 지원
  • 불공정 조항을 자동 탐지·시각화하여 의사결정의 공정성 강화
  • 개인별 계약서를 저장·관리할 수 있는 구조를 통해 지속적인 법률 관리 플랫폼 구현

3. 이론적 배경

3.1 자연어 처리 (NLP)

자연어 처리란 인간의 언어를 컴퓨터가 이해하고 분석할 수 있도록 하는 인공지능 기술입니다. 본 프로젝트에서는 계약서와 같은 비정형 텍스트 문서를 분석하여 조항의 의미를 파악하고 위험 요소를 식별하는 데 활용하였습니다.

3.2 파인튜닝 (Fine-Tuning)

사전 학습된 언어 모델은 일반적인 언어 패턴을 학습한 상태이므로, 법률 문서와 같은 특수 도메인에서는 한계가 존재합니다. 이에 따라 불공정 계약 조항 데이터셋을 기반으로 모델을 파인튜닝하여, 계약서 분석에 특화된 성능을 확보하였습니다.

4. 개발 방법 및 과정

4.1 개발 기간 및 환경

기간: 2025년 7월 31일 ~ 2025년 8월 2일
장소: 대전대신고등학교 프로젝트실

4.2 개발 절차

  • 1) 데이터 수집 및 전처리: 실제 근로계약서, 임대차 계약서 사례 조사 및 불공정 조항 선별, 텍스트 정제 및 라벨링
  • 2) AI 모델 파인튜닝: 사전 학습된 언어 모델을 기반으로 불공정 조항 판별 목적에 맞게 추가 학습 진행 (정확도 및 재현율 개선)
  • 3) 웹사이트 개발: Python(Flask) 기반 서버 구축, HTML/CSS UI 설계, 계약서 업로드 시 자동 분석 시각화 기능 구현
  • 4) 테스트 및 검증: 가상의 계약서를 입력하여 분석 결과 확인 및 실제 법률 자료와 비교 검증

5. 주요 기능 및 결과

5.1 AI 계약서 분석 기능

  • 사용자가 계약서를 업로드하면 AI가 자동으로 내용을 분석
  • 불공정 가능성이 있는 조항을 탐지하여 시각적으로 표시
  • 법률 지식이 없는 사용자도 핵심 위험 요소를 직관적으로 이해 가능

5.2 계약 이력 관리 기능

  • 개인별 계약서 저장 및 관리 전용 페이지 제공
  • 계약별 분석 결과를 누적 관리하여 단순 도구를 넘어 개인 법률 기록 관리 플랫폼으로 확장

  • IDE / 에디터: Visual Studio Code

7.1 결론

본 프로젝트를 통해 AI를 활용한 계약서 분석 웹 플랫폼을 실제로 구현함으로써, 법률 지식이 부족한 사용자도 계약의 공정성을 사전에 점검할 수 있음을 확인하였습니다. 이는 단순한 정보 제공을 넘어, 사회 문제 해결에 기여할 수 있는 IT 기반 서비스 모델로서의 가능성을 입증한 사례입니다.

  • 지자체·공공기관과의 협력을 통한 공공 법률 지원 플랫폼으로 확장
